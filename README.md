# Inaugural Lecture Date Booking System

This project is a web-based booking system developed using **PHP, MySQL, HTML, CSS, and JavaScript**.

## Features
- Staff registration and login
- Secure password hashing and reset via email (PHPMailer)
- First-come-first-serve booking system (no duplicate dates)
- Staff dashboard with booking status
- Admin dashboard to:
  - View all users and bookings
  - Edit, reset, or delete users
  - Export booking data to Excel
- Mobile-responsive design
- Session-based authentication

## Technologies Used
- PHP (Backend)
- MySQL (Database)
- HTML, CSS, JavaScript (Frontend)
- PHPMailer (Email service)

## Use Case
Designed for academic institutions to manage inaugural lecture date bookings efficiently and transparently.

## Installation
1. Clone the repository
2. Import the database SQL file
3. Configure database connection in `db.php`
